# 实验三：自顶向下的语法分析程序设计（LL(1)）

## 1. 文档依据

- `3 实验三_自顶向下的语法分析程序设计.doc`

## 2. 实验目标

- 正确计算 FIRST / FOLLOW 集。
- 构造 LL(1) 分析表。
- 对输入串执行 LL(1) 预测分析并输出步骤表。

## 3. 文档要求要点

- 输出内容应包含：
- FIRST 集
- FOLLOW 集
- LL(1) 分析表
- `步骤 / 分析栈 / 输入串 / 所用产生式` 过程表
- 建议测试表达式：`i+i*i`、`i*(i+i)`、`i+)i`（或数字表达式如 `1+2*3`）。

## 4. 目录结构

```text
ex3-自顶向下的语法分析程序设计
├── 3 实验三_自顶向下的语法分析程序设计.doc
├── LL1Parser.cpp
└── examples/
    └── grammar.txt
```

## 5. 代码任务

- 在 `LL1Parser.cpp` 中完成：
- `Sol_First()`
- `Sol_Follow()`
- `Build_LL1()`

## 6. 编译与运行

```bash
cd ex3-自顶向下的语法分析程序设计
clang++ -std=c++17 -O2 -o ll1_parser LL1Parser.cpp
./ll1_parser
```

运行时输入示例：

```text
examples/grammar.txt
1+2*3
```
