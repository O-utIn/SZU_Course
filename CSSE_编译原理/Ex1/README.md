# 实验一：高级语言及其文法

## 1. 文档依据

- `1 实验一_高级语言及其文法.doc`

## 2. 实验目标

- 加深对文法及其分类（0/1/2/3 型）的理解。
- 完成文法分类程序并通过给定测试数据验证结果。

## 3. 输入与输出要求（按文档）

- 输入：`examples/test*.in`（共 10 组），每行一条产生式。
- 处理：判断输入文法属于 0/1/2/3 型中的哪一类。
- 输出：在控制台打印每个用例的分类结果。

## 4. 代码任务

- 在 `program.cpp` 中补全：
- `is_type_3()`
- `is_type_2()`
- `is_type_1()`

## 5. 目录结构

```text
ex1-高级语言及其文法
├── 1 实验一_高级语言及其文法.doc
├── program.cpp
└── examples/
    ├── test0.in ... test9.in
```

## 6. 编译与运行

```bash
cd ex1-高级语言及其文法
clang++ -std=c++11 -O2 -o ex1 program.cpp
./ex1
```

或：

```bash
g++ -std=c++11 -O2 -o ex1 program.cpp
./ex1
```
