# 实验五：语法制导翻译和中间代码生成

## 1. 文档依据

- `5 实验五_中间代码生成程序设计.doc`

## 2. 实验目标

- 实现简单赋值语句的语法制导翻译。
- 将算术表达式转换为四元组（中间代码）序列。

## 3. 文档要求要点

- 输入：多条赋值语句，语句之间以 `;` 分隔。
- 处理：对右侧表达式拆解，引入临时变量，按优先级生成中间代码。
- 输出：四元组序列，如 `(op, arg1, arg2, result)`。
- 提供的 `ex.cpp` 为填空模板。

## 4. 目录结构

```text
ex5-中间代码生成程序设计
├── 5 实验五_中间代码生成程序设计.doc
├── ex.cpp
└── examples/
    ├── 1.txt ... 10.txt
```

## 5. 编译与运行

```bash
cd ex5-中间代码生成程序设计
clang++ -std=c++17 -O2 -o ex5 ex.cpp
./ex5
```

当前模板程序默认读取 `examples/10.txt`。
